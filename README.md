**Fresh. Organic. Interactive.**

MycoFresh is a premium web experience designed to showcase locally harvested oyster and milky mushrooms in Bangalore. Beyond just a landing page, this project is an exploration of modern web aesthetics and immersive 3D interaction.

##  Experience the Design

This project emphasizes a **visual-first approach**, combining organic textures with digital smoothness to create a unique brand identity for sustainable agriculture.

### Key Features

*   **Immersive 3D Elements**: deeply integrated 3D mushroom models that float and rotate, providing a tactile feel to the digital experience. Built with **React Three Fiber**.
*   **Glassmorphism UI**: A sleek, translucent interface that blurs the line between content and background, giving the site a modern, airy feel.
*   **Fluid Animations**: Powered by **Framer Motion**, every scroll and interaction is smooth and responsive, ensuring the user feels connected to the content.
*   **Dynamic Layouts**: A responsive design that adapts beautifully across devices, maintaining its premium look from desktop to mobile.

## Tech Stack

*   **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **3D Engine**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) & [Drei](https://github.com/pmndrs/drei)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)

##  Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/ash29032006/MycoFresh.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
